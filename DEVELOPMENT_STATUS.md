### 🚀 如何运行

1. **启动开发服务器**：
   ```bash
   cd customer-service-analyzer
   npm start
   ```

2. **访问应用**：
   - 打开浏览器访问: http://localhost:3000

3. **配置AI模型**：
   - 在左侧面板配置API参数
   - 选择服务商和模型
   - 保存配置

4. **上传文件**：
   - 拖拽或选择TXT文件
   - 等待文件处理完成
